CSS Concepts Tech-blog

In simple terms, classes and IDs are used in HTML and CSS to identify and style elements on a web page. Here are some best practices associated with using classes and IDs:

  1. Classes
    • Classes are used to group elements that share similar characteristics or styles. For example, you can have a class called "span" and apply it to multiple span element on your page.
    • Best practice is to use classes when you have multiple elements that need to be styled in the same way. It allows you to apply styles to multiple elements at once without repeating code.
    • Classes are not unique, meaning you can use the same class name on different elements throughout your page or even across different pages.

    Step by step guide:

    1. Add a class attribute with a class name in between the quotation mark to span element.
    2. In your style sheet file add CSS class attribute. Note to start with a "." followed by the class name.

  2. ID
    • IDs are used to uniquely identify a specific element on a web page. Each ID should only be assigned to one element.
    • Best practice is to use IDs when you need to target a specific element for a particular purpose. For example, you might have multiple input buttons on a page and you want to single out that one button to style it uniquely.
    • IDs are intended to be unique within the entire document, so it's not recommended to reuse the same ID on multiple elements.

    STEP by STEP guide:

    1. Add an Id attribute with an Id name in between the quotation mark to input buttons.

    2. In your style sheet file add Id attribute. Note to start with "#" followed by the Id name.
